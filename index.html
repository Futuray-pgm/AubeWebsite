  1. Download Luanti on https://luanti.org/downloads/
  2. Install and run the game engine
  3. Go to the Join Game tab
  4. Type auberpg.minetest.landin the address field
  5. Check the port is 3000
  6. Click Create Acount, choose a name and a password
  7. Now, you can connect and play!
